## Instalacion

Descargue desde GitHub

`git clone https://github.com/jsola02/Curso_CSS-Flexbox.git`

abrir proyecto, windows

`cd Curso_CSS-Flexbox`

instalar paquetes requeridos de npm (node_modelues)

`npm install`

ejecutar el proyecto

`npm run dev`
